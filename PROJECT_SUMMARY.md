5. Click bookmark on any website

## üÜò Help & Troubleshooting

### Common Issues

**Q: Nothing happens when I click Start?**
A: Make sure to click on your target first. Try lower speeds. Use Python version for better results.

**Q: Python version won't run?**
A: Install pynput: `pip install pynput`. Try administrator/sudo mode.

**Q: F6 hotkey not working?**
A: Use web version instead. Close other programs. Run Python as administrator.

**Q: Clicks too slow?**
A: Increase CPS value. Close unnecessary programs. Use Python version.

- See GETTING_STARTED.md for step-by-step guide
- See README.md for full documentation
